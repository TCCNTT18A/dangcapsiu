Giới thiệu về HTML

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu tiêu chuẩn để tạo ra các trang web. HTML mô tả cấu trúc của một trang web.

Các phần tử HTML được biểu diễn bằng các thẻ (tags). Trình duyệt web không hiển thị các thẻ HTML mà sử dụng chúng để hiển thị nội dung.

<!DOCTYPE html>
<html>
<head>
  <title>Tiêu đề trang</title>
</head>
<body>
  <h1>Đây là tiêu đề</h1>
  <p>Đây là một đoạn văn.</p>
</body>
</html>
Tìm hiểu thêm

Các thẻ HTML phổ biến

  • <h1> - <h6>: Tiêu đề (heading)
  • <p>: Đoạn văn (paragraph)
  • <a>: Liên kết (hyperlink)
  • <img>: Hình ảnh
  • <ul>, <ol>, <li>: Danh sách
  • <div>: Phần tử khối (block)
  • <span>: Phần tử nội tuyến (inline)

Các thuộc tính thường dùng

  1. id: Định danh duy nhất cho phần tử
  2. class: Xác định lớp CSS cho phần tử
  3. src: Đường dẫn đến hình ảnh hoặc tệp
  4. href: Đường dẫn cho liên kết
  5. alt: Văn bản thay thế cho hình ảnh

Cấu trúc một trang HTML

Một tài liệu HTML hợp lệ luôn bắt đầu với khai báo <!DOCTYPE html>.

Các phần chính của trang HTML:

  1. Khai báo DOCTYPE: Xác định phiên bản HTML
  2. Thẻ <html>: Phần tử gốc của trang
  3. Thẻ <head>: Chứa thông tin meta, tiêu đề, liên kết CSS
  4. Thẻ <body>: Chứa nội dung hiển thị của trang

HTML5 cũng giới thiệu các thẻ ngữ nghĩa như:

  • <header>, <footer>
  • <nav>, <main>, <article>
  • <section>, <aside>
